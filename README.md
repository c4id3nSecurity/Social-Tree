# Social Tree App

## What's next? How do I start the project?

- `git clone https://github.com/c4id3nSecurity/Social-Tree.git`
- `cd Social-Tree`
- `npm install`
- `npm dev`

## Technologies used in the project

- [Next.js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com)
